## 关键取舍

- 不引入工作流引擎、数据库、消息队列和多层 Agent 编排器。
- SubAgent 仅提供有限 researcher/reviewer 能力，不继承 MCP，避免权限和上下文无边界扩张。
- Goal 复用 Plan 存储；自动 RAG 使用本地词法检索，显式检索才按需使用 Embedding。
- MCP 协议与 Agent Loop 继续交给 Agents SDK，MimiAgent 只维护配置、生命周期和展示层。
