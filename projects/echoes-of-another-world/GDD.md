# 异界残响 (Echoes of Another World)

## 类型
上帝视角 2D 像素 · 开放世界 ARPG

## 世界观
异世界召唤 × 时空崩坏 × 魔法朋克

## 参考
Re:Zero 的异世界沉浸感 + GTA 的自由度 + 以撒的结合的俯视角

---

## 开篇设定

主角是地球的普通社畜，深夜点击了一个神秘对话框后被传送到异世界「菲尔迦纳」。召唤仪式失败——仪式本应召唤大贤者对抗【虚噬】，却召来了一个普通人。失败的仪式在主角体内留下了【残响】——触碰时空裂隙的能力。没有使命，没有系统面板，自生自灭。

## 世界 · 菲尔迦纳大陆

大陆正被【虚噬】蚕食，时空像碎玻璃一样被重新拼接。四大区域无缝连接，按距离自动流式加载。

| 区域 | 风格 | 地标 |
|------|------|------|
| 王冠平原 | 中世纪城镇+麦田+骑士营地 | 王都阿斯特拉、召唤遗迹 |
| 魔女森林 | 幽暗密林+发光孢子树+巨兽骸骨 | 魔女宅邸、禁忌图书馆 |
| 裂隙海岸 | 破碎岛屿+搁浅古代飞船+盐晶沙漠 | 自由城邦卡萨布兰、时空裂隙核心 |
| 终焉山脉 | 活火山+龙族废墟+浮空碎片 | 龙脊圣所、虚噬裂隙入口 |

## 核心设计原则

- 主线可全程忽略，没有"不推主线就解锁不了"的限制
- 任何NPC可对话、攻击、偷窃（有后果）
- 不设任务失败——只留不同后果
- 自由混合多种玩法：赏金猎人、法师、盗贼、商会、雇佣兵
- 大部分区域不做等级锁

## 魔法系统 · 残响

- **33个残响印记** 散布在世界各处（遗迹、Boss、隐藏任务、裂隙）
- 同时可装备 4 个印记，每个 = 一个主动能力
- **三系**：时序（操控时间流速）、空间（瞬移/重力/裂隙切割）、记忆（读取NPC记忆/召唤幻影）
- **共鸣组合**——特定印记搭配触发隐藏技：
  - 时序+空间 →「因果断裂」（短时无敌+伤害翻倍）
  - 记忆+记忆 →「死者回响」（召唤区域中最后死去的强者为你而战）
- 使用残响会被「虚空监视者」察觉——用多了会有不可名状的存在来找你

## 势力系统

声望网络制，加入不锁死：

| 势力 | 追求 | 敌对后果 |
|------|------|----------|
| 王国骑士团 | 秩序、保护平民 | 通缉（1-3星） |
| 魔女集会 | 研究残响、对抗虚噬 | 禁术反噬、违约惩罚 |
| 自由城邦商会 | 金钱、裂隙遗物交易 | 暗杀+价格封杀 |
| 虚空守望者 | 抹除触碰时空之人 | 不可贿赂逃跑，只能战斗或隐藏 |

## 动态剧情系统

**主线**（约20小时）：三方势力围绕虚噬的博弈——重塑世界派 vs 封印派 vs 逃亡派。主角的存在本身打破平衡。多结局。

**支线不设失败，只设后果。**
- 商队护卫任务中途离开 → 商队全灭，NPC关系永久改变
- 钱包被偷 → 追上打死 / 放过得情报 / 跟踪到盗贼公会 / 无视

**动态事件**：旅行中随机遭遇——裂隙兽潮、帝国逃兵、行商求救。靠近触发，不靠近就发生。

## 追随者系统

- 约15个可发展关系的独特追随者
- 在世界中遇到、救过、或被救过——不是在菜单里招募
- 好感度影响战斗表现、秘密分享、生死抉择、背叛条件
- 每人有独立任务线和结局，部分彼此冲突
- 可带1个出门，其余在指定城镇待命

## 成长体系

无职业等级，只有熟练度：

| 行为 | 提升 |
|------|------|
| 剑系攻击 | 剑术 → 伤害+攻速 |
| 时序残响 | 时序掌控 → 冷却缩短 |
| 翻滚/闪避 | 敏捷 → 无敌帧延长 |
| 说服/欺骗 | 话术 → 特殊对话选项 |
| 烹饪/炼药/锻造 | 制作技能 |

装备：武器+护甲+饰品+残响容器（决定印记槽位数量）

## 世界记忆系统

- NPC死亡后其相关人物会在数月后寻仇（除非走和解任务线）
- 残响印记出售后可能在新Boss身上重新出现——你种下的因果
- 帮助村庄的事迹会在路过行商口中流传

## 美术风格

- 角色/NPC：16×16 像素
- 场景Tile：32×32，每个区域独特色板
- 2D动态光照：灯火、残响蓝紫辉光、裂隙红色脉动
- UI极简风

## 技术方案

| 模块 | Unity方案 |
|------|-----------|
| 引擎版本 | Unity 2022 LTS |
| 开放世界流式加载 | 分块Tilemap + 异步加载，9宫格激活 |
| 2D光照 | URP 2D Renderer + Light2D |
| NPC行为 | 状态机 + 有限行为树 |
| 事件系统 | ScriptableObject Event Channel |
| 存档 | JSON + 增量保存 |
| 输入 | Input System（键鼠优先，手柄适配） |
